# Smart QR-Based Visitor Approval System

A mini project developed using HTML, CSS, JavaScript, PHP, and MySQL.

## Features
- QR-based visitor entry form
- Owner approval and rejection
- Guard dashboard for monitoring
- Auto-rejection during owner vacation mode
- Email notification support
- Progressive Web App (PWA) with offline support

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- PHPMailer

## Modules
- Visitor Module
- Owner Module
- Guard Module

## How to Run
1. Install XAMPP
2. Place the project folder inside `htdocs`
3. Start Apache and MySQL
4. Import the database file into phpMyAdmin
5. Open the project in browser

## Note
Sensitive credentials such as email passwords and database passwords are not included in this repository.
